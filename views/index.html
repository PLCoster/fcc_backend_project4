Exercise Tracker

Create a New User

POST /api/users

Add Exercise

POST /api/users/:_id/exercises

NOTE: This is a DEMO site. Created Users and Exercises are deleted after 24hrs.

Example Usage:

To create a new User:
  • Submit a username using the 'Create a New User' form above.
  • Alternatively, submit a POST request to /api/users with a body containing a url encoded field of 'username' (e.g. 'username=trackerUser')
To view all Users:
  • Send a GET request to /api/users
    /api/users
To add an Exercise to a User Account:
  • Submit an exercise using the 'Add Exercise' form above.
  • Alternatively, send a POST request to /api/users/<USER_ID>/exercises with a valid USER_ID, and a body containing url encoded fields of:
    • 'description' (string)
    • 'duration' (integer)
    • 'date' (optional, YYYY-MM-DD)
  • For example:
    • 'description=jogging&duration=30&date=2022-07-16
To view a User's Exercise Log:
  • Send a GET request to /api/users/:_id/logs?[from][&to][&limit]
    where:
    • [] denote optional filter query parameters
    • from, to = dates (yyyy-mm-dd); limit = number
  • For example:
    • GET all logs for User 'trackerUser':
      /api/users/6309708985ce1de85952ada6/logs
    • GET 10 most recent logs for User 'trackerUser':
      /api/users/6309708985ce1de85952ada6/logs?limit=10
    • GET up to 5 most recent logs between 1st Jan 2021 and 31st Dec 2022 for User 'trackerUser':
      /api/users/6309708985ce1de85952ada6/logs?limit=5&from=2021-01-01&to=2021-12-31

Example Output:

When creating a new User, or requesting all Users, a single User object or array of User objects will be returned in JSON format, respectively. The User objects have the following form:

{
  • "username": "trackerUser",
  • "_id": "6309708985ce1de85952ada6",
}

When adding an Exercise for a User, an Exercise Object in JSON format is returned with the form:

{
  • "username": "trackerUser",
  • "_id": "6309708985ce1de85952ada6",
  • "description": "jogging",
  • "duration": "35",
  • "date": "Fri Aug 26 2022"
}

When requesting the exercise logs for a User, a Log Object in JSON format is returned with the form:

{
  • "username": "trackerUser",
  • "_id": "6309708985ce1de85952ada6",
  • "filters": {
    • "from": "2021-01-01" || "any",
    • "to": "2021-12-31" || "any",
    • limit: 2 || "all"
    • },
  • "count": 2,
  • "log": [
    • {"description":"jogging","duration":30,"date":"Sat Dec 25 2021"},
    • {"description":"swimming","duration":60,"date":"Fri Dec 24 2021"}
    • ]
}
